# SIMPLE-AI

SIMPLE-AI is a conversation-first full stack web app concept focused on helping non-technical users create digital solutions through natural language.

## Vision

The system should guide a user through a conversational flow to:

- collect business context
- ask follow-up questions
- define requirements
- plan the stack and tooling
- iterate with the user until the solution is clear

## Initial Direction

- Conversation-driven product building
- Full stack web application
- Focus on reducing technical complexity
- Target audience: service businesses and non-technical users
- Early references: Convo.ai and Oracle

## Notes

This repository is the starting point for the project.
