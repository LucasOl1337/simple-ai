# Mobile First Gallery

Familia de referencia para observar como bons sites se comportam em pouca largura.

## O que procurar

- Hero legivel em 375px.
- CTAs grandes e claros.
- Cards com bom espacamento.
- Ordem de secoes que faz sentido em scroll curto.

## O que evitar

- Conteudo comprimido.
- Titulos que quebram mal.
- Blocos horizontais que dependem de desktop.
