# Catalogs Gallery

Familia de referencia para paginas com volume medio ou alto de itens.

## O que procurar

- Grid consistente.
- Categorias ou agrupamentos claros.
- Ritmo visual simples para escaneabilidade.
- CTA sempre acessivel.

## O que evitar

- Repeticao visual cansativa.
- Preco ou acao escondidos.
- Cards pequenos demais no mobile.

## Relacao com perfis atuais

- `catalogo-moderno`
- `site-builder-core`
