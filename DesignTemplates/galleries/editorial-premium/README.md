# Editorial Premium Gallery

Familia de referencia para marcas e profissionais que pedem sofisticacao e curadoria.

## O que procurar

- Titulos fortes.
- Espaco negativo bem usado.
- Ritmo visual elegante.
- Poucos elementos, cada um com intencao.

## O que evitar

- Conceito acima da clareza.
- Menosprezar CTA por querer parecer artistico.
- Tipografia rebuscada demais para o publico.
