# Local Services Gallery

Familia de referencia para servicos locais com foco em confianca, proximidade e contato rapido.

## O que procurar

- Hero direto com CTA visivel.
- Prova de confianca cedo: processo, especialidades, atendimento, area.
- Contato repetido sem poluir a pagina.
- Linguagem visual humana, nao corporativa demais.

## O que evitar

- Visual de software.
- Cards vazios sem detalhe util.
- Excesso de elementos abstratos.

## Relacao com perfis atuais

- `servico-local-confianca`
- `site-builder-core`
