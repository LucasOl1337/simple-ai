# Warm Artisanal Gallery

Familia de referencia para negocios com apelo humano, sensorial e artesanal.

## O que procurar

- Fotografia proxima.
- Materiais e cores quentes.
- Tipografia com mais personalidade, quando fizer sentido.
- Sensacao de cuidado e autenticidade.

## O que evitar

- Rusticidade caricata.
- Excesso de textura e pouca legibilidade.
- Visual gourmet genérico sem clareza de oferta.
