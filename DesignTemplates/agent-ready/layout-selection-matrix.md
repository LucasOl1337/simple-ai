# Layout Selection Matrix

## Matriz principal

| Sinais do briefing | Layout recomendado | Observacoes |
|---|---|---|
| `primary_cta` unico + poucos servicos | Landing curta de conversao | Hero dominante, beneficios e CTA final |
| `scope=local` + contato por WhatsApp | Institucional local com confianca | Contato no topo, processo simples, FAQ opcional |
| `content_volume=medio` | Institucional local ou catalogo leve | Escolher conforme variedade e preco |
| `content_volume=alto` | Catalogo escaneavel | Categorias e grid consistentes |
| `brand_tone=serio/profissional` | Clean profissional ou editorial sobrio | Evitar efeitos chamativos |
| `brand_tone=acolhedor/familiar` | Calor humano / amigavel acessivel | Fotos e proximidade importam |
| `has_media=true` + negocio sensorial | Pagina acolhedora com imagem dominante | Garantir CTA e texto objetivo |

## Ajustes por CTA

| CTA principal | Prioridade visual |
|---|---|
| WhatsApp | Botao visivel no hero e no final |
| Ligar | Telefone clicavel e destaque de rapidez |
| Pedir orcamento | Hero objetivo + prova de confianca |
| Agendar | Hero + processo + disponibilidade ou contexto |
| Comprar | Grid forte, categorias e preco quando houver |

## Ajustes por volume de conteudo

| Volume | Estrategia |
|---|---|
| 1-5 itens | Blocos curtos ou cards grandes |
| 6-15 itens | Grid simples com grupos ou destaque de principais |
| 15+ itens | Catalogo por categorias, filtros visuais leves ou seccoes agrupadas |
