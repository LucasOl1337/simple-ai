# GitHub Notes

## O que o GitHub entrega melhor para esta biblioteca

- anatomia de HTML e CSS
- exemplos pequenos por nicho
- implementacoes sem build complexo
- variacoes comparaveis dentro de uma mesma pagina

## O que o GitHub entrega pior

- nivel medio de direcao visual costuma ser inferior ao de galerias curadas
- varios projetos didaticos usam estilos datados
- nem sempre ha boa adequacao para mobile ou para pequenos negocios reais

## Leituras uteis observadas

### CopilotKit showcase variations

- URL: `https://github.com/CopilotKit/CopilotKit/blob/main/showcase/shell/public/designs/front-door-v2.html`

- URLs:
  - `https://github.com/Ayushparikh-code/Web-dev-mini-projects/blob/main/Product%20Landing%20Page/index.html`
  - `https://github.com/HuXn-WebDev/HTML-CSS-JavaScript-100-Projects`
- Aprendizado: mais uteis como anatomia minima do que como referencia final de design.

## Aplicacao pratica no SIMPLE-AI

Usar GitHub para:

- estudar estrutura de secoes
- observar implementacoes leves
- reaproveitar ideias de semantica e ordem de blocos

Nao usar GitHub como unica fonte de direcao visual final.
