# Section Patterns

## Hero

- titulo curto
- subtitulo objetivo
- CTA principal acima da dobra
- imagem ou apoio visual coerente com o segmento

## Oferta / Servicos

- lista curta para poucos itens
- grid consistente para volume medio
- agrupamento por categoria para volume alto

## Prova / Confianca

- processo simples
- especialidades
- perguntas frequentes
- contexto humano

## CTA final

- repetir a mesma acao principal
- reduzir friccao
- evitar apresentar uma acao concorrente nova no final

## Footer

- simples
- nome do negocio
- contato resumido ou area atendida quando fizer sentido
