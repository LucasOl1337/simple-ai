# HTML Examples

## Casos iniciais observados

- Landing pages em HTML puro com hero, oferta e CTA.
- Exemplos de nicho como construction, food e startup.
- Estruturas uteis para o builder atual, que produz um unico arquivo HTML.

## O que extrair

- semantica clara de secoes
- relacao entre hero e navegacao
- organizacao de cards e grids
- estrategia de CTA em paginas simples

## O que nao extrair automaticamente

- visuais antigos ou datados
- excesso de efeitos ou carrosseis
- dependencias pesadas
