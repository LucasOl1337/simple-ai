# Curated Repos

## Lista inicial

### SHSFWork/awesome-inspiration

- URL: `https://github.com/SHSFWork/awesome-inspiration`
- Utilidade: diretorio de galerias, components e fontes de inspiracao.
- Melhor uso: expandir referencias futuras da biblioteca.

### CopilotKit/CopilotKit

- URL: `https://github.com/CopilotKit/CopilotKit/blob/main/showcase/shell/public/designs/front-door-v2.html`
- Utilidade: ver variacoes visuais explicitamente separadas em uma mesma pagina.
- Melhor uso: estudar comparacao entre hero terminal e hero editorial.

### you-dont-need/You-Dont-Need-JavaScript

- URL: `https://github.com/you-dont-need/You-Dont-Need-JavaScript/blob/master/Landing%20Page/index.html`
- Utilidade: exemplo simples de landing em HTML puro.

### Ayushparikh-code/Web-dev-mini-projects

- URL: `https://github.com/Ayushparikh-code/Web-dev-mini-projects/blob/main/Product%20Landing%20Page/index.html`

### HuXn-WebDev/HTML-CSS-JavaScript-100-Projects

- URL: `https://github.com/HuXn-WebDev/HTML-CSS-JavaScript-100-Projects`
- Utilidade: exemplos de paginas por nicho como construction e food.
- Melhor uso: observar adaptacao de tema para segmento.

## Observacao

Muitos repositórios de landing page em GitHub sao didaticos e nem sempre representam o melhor nivel de design. Eles servem mais para anatomia, semantica e padroes simples de implementacao do que para direcao visual final.
