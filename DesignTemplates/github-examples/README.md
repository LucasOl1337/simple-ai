# GitHub Examples

Esta area reune exemplos tecnicos e anatomia de implementacao.

## Objetivo

- entender estruturas reais de HTML e CSS
- observar ordem de secoes
- estudar composicao de hero, grids e blocos de CTA
- encontrar exemplos simples que conversem com o builder atual de HTML unico

## Como usar

- priorizar exemplos pequenos e legiveis
- extrair estrutura, nao identidade visual
- evitar copiar projetos inteiros sem necessidade
