Sobre

Moda com atendimento próximo.

A Moda Viva é uma loja de roupas pensada para quem gosta de escolher com tranquilidade. O foco é mostrar o catálogo, destacar as novidades e manter um canal simples para pedidos pelo WhatsApp.

O atendimento funciona como uma ponte entre a loja física e a compra online: você vê as peças, tira dúvidas e combina os próximos passos diretamente.
