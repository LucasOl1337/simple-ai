Sobre

Uma loja feita para atender perto e online.

A Moda Viva é uma loja de roupas que atende clientes pela internet e também em loja física. O site funciona como uma vitrine direta: mostra o catálogo, destaca novidades e facilita o início do pedido.

Como muitos clientes já chegam pelo Instagram e pelo WhatsApp, a página mantém esses canais como caminho principal de contato.
